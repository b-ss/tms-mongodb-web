需要在环境变量中设置是否开放复制集合功能。

TMW_REPLICA_COLLECTION

| 内置字段     | 说明                            | 必填 | 类型      |
| ------------ | ------------------------------- | ---- | --------- |
| \_\_src      |                                 | 是   | object    |
| \_\_src.db   | 文档所属数据库名称（sysname）。 | 是   | string    |
| \_\_src.coll | 文档所属集合名称（name）。      | 是   | string    |
| \_\_src.id   | 文档原始 id（\_id）。           | 是   | ObjectId  |
| \_\_src.time | 最近一次复制时间。              | 是   | 13 位整型 |
